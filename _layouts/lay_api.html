Application programming interface (API)

The API application allows to browse and download for further processing the full range of data and metadata on Sustainable Development Indicators. The lis below presents vailable web services provided as part of the API.

For API security reasons, quantitative limits are imposed on the number of requests. For unauthenticated requests, the rate limit allows for up to 60 requests per hour. For API requests using Basic Authentication or OAuth, you can make up to 5000 requests per hour.
